# 数据更新测试工具 - MVP版本

最简单的数据库更新工具，用于测试前端实时刷新。

## 快速开始

```bash
cd test
conda activate breath
python test_update_data.py --mode quick
```

## 功能说明

### 每次更新内容

1. **波形数据**（实时刷新）
   - 呼吸波形：200个点
   - 呼吸环形图：X/Y各1000个点
   - 心率波形：200个点
   - SCG波形：1000个点

2. **历史记录**（批量更新）
   - 呼吸历史：更新最新200条记录
   - 心率历史：更新最新200条记录

### 运行模式

**快速测试（推荐）**
```bash
python test_update_data.py --mode quick
```
单次更新所有数据，查看详细输出。

**持续更新**
```bash
# 每2秒更新一次，持续60秒
python test_update_data.py --mode continuous

# 每5秒更新一次，持续2分钟
python test_update_data.py --mode continuous --interval 5 --duration 120

# 无限运行（Ctrl+C停止）
python test_update_data.py --mode continuous --interval 3 --duration 0
```

## 测试流程

1. 确保Flask后端服务器正在运行
2. 打开前端页面
3. 运行测试脚本
4. 观察前端页面数据变化

## 注意事项

- 所有数据使用UPDATE而非INSERT
- 每次更新会打印详细统计信息
- 波形数据使用随机相位生成，确保每次不同
- 历史记录保持时间戳更新
